Controles:

Interagir com o menu/Colocar tropa: Botao esquerdo do Mouse 
Cancelar Colocação de tropa: Botao direito do Mouse/ESQ
Redirecionar tropa para a esquerda: <-
Redirecionar tropa para a direita: ->
W A S D : Movimentação de camera
NUMPAD 1: Construir tropa guerreiro
NUMPAD 2: Construir tropa arqueiro
NUMPAD 3: Construir tropa druida
*: Finalizar o jogo com Vitoria

Para a instalação: 
Como esse arquivo foi feito em windows para linux pode ser necessario dar a permição para o arquivo em build conseguir instalar o jogo. Entao tente "chmod +x MedievalTowerDefence.x86_64" no terminal
